# AttriSense AI – Employee Attrition Prediction System

## Overview

AttriSense AI is a Machine Learning web application that predicts whether an employee is likely to stay or leave an organization based on employee details.

The application is built using Flask and provides a clean dashboard with prediction history, charts, PDF report generation, and an intuitive user interface.

---

## Features

- Employee Attrition Prediction
- Interactive Web Dashboard
- Prediction History
- Dashboard Cards
- Pie Chart Visualization
- Risk Level Detection
- PDF Report Download
- Date & Time Tracking
- Clear History Option
- Responsive User Interface
